# Getting Started

## Installation
```bash
$ yarn
```

## Run the app
```bash
# development
$ yarn dev

# production mode
$ yarn build
$ yarn start
```

## Test
```bash
# unit tests
$ yarn test

# watch changes and re-run tests
$ yarn test:watch

# test coverage
$ yarn test:coverage
```
Open /coverage/lcov-report/index.html to see report in browser
